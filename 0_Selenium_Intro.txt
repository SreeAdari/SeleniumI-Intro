Introduction to Selenium :
============================================

1.Selenium is an open source test Automation tool
2.key components of selenium are Selenium WebDriver, Selenium IDE,Selenium Grid
3.Exclusively used for testing Web Applications on multiple browsers like Firefox,chrome,IE,Edge
4.Selenium supports multiple operatig systems like Windows, Macos,Liinux
5.Selenium supports multiple programming langauges like Java,Python,C#,Javascript,Ruby
